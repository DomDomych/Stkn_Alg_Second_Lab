# **Лабораторная Работа №2**  
"Транспорт (Bus Route)"

Работа реализует маршрут автобуса на двусвязном списке. Список хранит в себе Класс Остановок. Остановка состоит из ее названия, координат, времени до следующей остановки, а также ссылок на следующую и предыдущую остановки.

## **Реализованные функции:**

**Обязательная часть:**

Добавление остановки в маршрут;  
Расчет общего времени маршрута;    
Определение остановки, на которой будет автобус через N переходов;    
Построение маршрута в обратном порядк;    

**Вариативная часть:**  

Создание кольцевого маршрута;  
Поиск остановок, до которых автобус доедет за заданное время;  
Расчет времени прибытия автобуса на каждую остановку;  
Формирование подробного отчета маршрута с сохранением в текстовый файл;  
Вывод маршрута в виде таблицы с выравниванием колонок;  
Сохранение маршрута в текстовый файл, где каждая строка содержит Base64-кодированную информацию об остановке;   

Двусвязный список был выбран, потому что таким образом удобно строить прямой и обратный маршруты, а также реализовывать кольцевой маршрут  

### **Сборка происходит с помощью команд:**  

cmake -S . -B build  
cmake --build build  

### **Программа запускается командой:**  

./build/bus_route  

### **Тесты запускаются командой:**  

./build/route_tests  

Тесты проверяют общую длительность маршрута, функции After_N, ByTheTime,Stopping,Base64 кодировку и правильную запись в файл  

Обработаны граничные случаи: пустой маршрут, отрицательное N в After_N, неверное время,выход за границы, кольцевой маршрут  

Программа может создавать файлы: route_tab.txt, route_report.txt, route_base64.txt

Также тесты могут создавать временные файлы
